We can now query for all exons that belong to protein-coding genes and group them by transcript. By summing the width of the exons we get the length of the spliced transcripts.

mm_exons <- exonsBy(mm_ensdb,
  by = "tx",
  filter = ~gene_biotype == "protein_coding"
)

Few transcripts are very long which skews the distribution and makes the plot not very informative. The longest transcript (ENSMUST00000020488) is 123’179 bps long. Interestingly, the last exon - exon 4 - makes up most of the transcript and is 122’802 bps long (source). This can also be seen in the diagram below from the ensembl website. Filled boxes mark translated regions and unfilled boxes show UTRs.

The coding region is only 558 bps long which seems reasonable.

cds <- cdsBy(mm_ensdb, by = "tx", filter = TxIdFilter("ENSMUST00000020488"))
